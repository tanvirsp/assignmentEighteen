# Assignment Eighteen
## This is an Ecommerce Backend Sever for Ostad Assignment
### Security Middleware List
* express-mongo-sanitize
* express-rate-limit
* helmet
* hpp
* jsonwebtoken
___

### Functionality in used
* User Login/Logout
* Login by OTP
* User Can Create Profile, Update & Read
* Add To Cart
* Add Product 
* Product Details
* WishList
